# Stock Insight K 线图重新设计文档

## 1. 目标

用原生 Canvas 实现同花顺风格的 K 线图，移除 TradingView Lightweight Charts，换用自研实现。

## 2. 技术方案

### Canvas 分层架构

```
┌─────────────────────────────────────────┐
│  信息框层 (十字光标悬浮显示)              │
├─────────────────────────────────────────┤
│  主图 (K线 + 均线) — Canvas            │
├─────────────────────────────────────────┤
│  成交量副图 — Canvas                   │
├─────────────────────────────────────────┤
│  MACD 副图 — Canvas (可选)            │
├─────────────────────────────────────────┤
│  KDJ/BOLL/RSI 副图 — Canvas (可选)    │
└─────────────────────────────────────────┘
```

- 所有副图共享同一个时间轴（X 轴）
- 鼠标滚轮缩放、拖拽平移（全局联动）
- 信息框跟随十字光标显示

### React 集成方式

- `useRef` 获取各 Canvas DOM
- `useEffect` 初始化图表、绑定事件
- `useState` 管理 K 线数据、缩放、平移状态
- 数据变化时调用 Canvas 重绘

### 数据格式

```typescript
interface KLineData {
  date: number;      // yyyymmdd 格式，如 20260331
  open: number;
  high: number;
  low: number;
  close: number;
  volume: number;    // 成交量
  amount?: number;   // 成交额（可选）
}
```

均线（MA5/10/20/30/60）由计算得出，不需要额外存储。

## 6. 组件结构

```
components/
  HQChart.tsx         # 主组件，管理所有 Canvas 层
  ChartCanvas.tsx     # 单个 Canvas 绘图区域
  CrosshairOverlay.tsx  # 十字光标 + 信息框
  PriceAxis.tsx       # 右侧价格轴
  TimeAxis.tsx        # 底部时间轴

lib/
  indicators.ts        # 指标计算（MA, MACD, KDJ, BOLL, RSI）
  chartRenderer.ts    # Canvas 渲染核心
```

8. 实现 MACD/KDJ/BOLL/RSI 副图
